A web application that uses AI to transcribe spoken audio into text in real-time.

## Overview

This application allows users to record audio through their browser, and uses AI-powered speech recognition to automatically convert the spoken words into written text. The transcription includes speaker diarization, identifying and separating different speakers in the conversation.

## Features

- üé§ **Audio Recording**: Record audio directly from your browser
- ü§ñ **AI Transcription**: Automatic conversion of speech to text using AI

## How It Works

1. Click the record button to start recording
2. Speak into your microphone
3. Stop recording when finished
4. The AI processes your audio and displays the transcribed text
5. Each speaker's words are clearly labeled and separated

## Technology

- **Frontend**: React with Vite
- **Backend**: Node.js with Express
- **AI Service**: ElevenLabs Speech-to-Text API (can be changed)
